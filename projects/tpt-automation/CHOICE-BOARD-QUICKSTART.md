# Choice Board Generator - Quick Start

**Goal**: Generate 5 professional choice boards for TPT in under 5 minutes

---

## Step 1: Generate Boards (30 seconds)

```bash
python scripts/generators/choice_board_generator.py
```

**Output**: 5 PowerPoint files in `output/choice-boards/`

---

## Step 2: Review Files (2 minutes)

Open each .pptx file and verify:
- Title slide looks professional
- 3x3 grid displays correctly
- Rubric page is clear

---

## Step 3: Optional Customization (2 minutes)

